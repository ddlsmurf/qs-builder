enum_with_sep(enumerable, last_sep = ' and ', sep = ', ') {|*arguments| ...}

if enumerable is nil, return nil if enumerable has any items, concatenate them, separating them using sep and last_sep

sep
separator to use between all items except the last and before-last
last_sep
is the separator to use between the before-last and last items
when_not_empty(value) {|value| ...}

yields value if its to_s has length

wrap(text = nil, prefix = "<nowiki>", suffix = "</nowiki>") {|text| ...}

wrap the text in prefix and suffix if its not empty. if a block is given, its return value is used as the text if non false

wrap_tag(text, tag = "nowiki", &block)

wraps in html like tags. does not escape text.
